✨ For Jemi

A small digital birthday gift built with love.

This project is an interactive Star Jar inspired by lucky-star jars, affirmation jars, and handwritten birthday notes. Inside the jar are colorful folded stars, each containing a different message of encouragement, faith, comfort, love, or celebration.

At the top of the jar sits a special birthday star, waiting to be opened.

💜 The Idea

Instead of giving a traditional card, this project creates a digital experience where each star holds a message.

The goal was to make something that feels:

Personal
Soft
Meaningful
Comforting
Faith-filled
A little magical

Each star represents a different emotion or reminder and reveals a unique note when opened.
